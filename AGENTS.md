## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues for this repository (use `gh` CLI). See `docs/agents/issue-tracker.md`.

### Triage labels

We use the default five-label triage vocabulary (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses a single-context domain layout at the root (`CONTEXT.md` + `docs/adr/`). See `docs/agents/domain.md`.
